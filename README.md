# 🎧 Podwise – Podcast Recommendation System

Podwise is a full-stack podcast application built using Flutter and Python Flask that allows users to discover, listen, and manage podcasts efficiently.

It provides personalized podcast recommendations based on user interests and supports real-time synchronization between frontend and backend.

## 🚀 Key Highlights
- Personalized podcast recommendation system  
- Advanced search with filters (category, duration, rating)  
- Smart library management (Subscribed, Saved, Downloaded)  
- Auto-save episodes on play with progress tracking  
- High-quality audio player with speed control and skip options  
- Real-time data synchronization  

## 🛠️ Tech Stack
Flutter (Dart) | Python (Flask) | Pandas | REST API  

## 📌 About Project
This project focuses on solving the problem of discovering relevant podcast content from a large dataset.  
It demonstrates the implementation of a recommendation system along with a complete full-stack application.
